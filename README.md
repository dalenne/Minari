# Projet : Réseau Social (Internet et Outils)

## Contexte

Ce projet a été réalisé dans le cadre de la matière "Internet et Outils" en L1.
Le but du cours était de nous apprendre les bases du développement web avec HTML, CSS, PHP et SQL.

## Sujet

Le projet consistait à développer un réseau social en binôme, avec un certain nombre de fonctionnalités imposées.

## Fonctionnalités

Voici les principales fonctionnalités mises en place :

* Design responsive (adapté à la taille de la fenêtre)
* Interface inspirée de Twitter

### Gestion des utilisateurs

* Création de compte
* Connexion/Déconnexion
* Deux types de comptes : utilisateur classique et administrateur qui peuvent être en public ou en privé

### Contenu

* Publication de posts permanents
* Publication de posts éphémères
* Fil d’actualité basé sur les abonnements

### Interactions

* Messagerie privée (MP)
* Rechercher un profil par son pseudo ou son surnom

### Profil

* Consultation du profil
* Modification des informations :
    * Pseudonyme
    * Photo de profil
    * Bannière
    * Biographie
    * Visibilité du profil

### Modération

* Signalement de posts
* Outils de modération pour les administrateurs
* Suppression de contenus signalés

## Ce que j’ai appris

Ce projet m’a permis de :

* Comprendre le fonctionnement d’une application web complète
* Utiliser une base de données SQL avec PHP
* Structurer un projet web
* Travailler en binôme

## Remarques

C’est un projet assez ancien, donc certaines parties pourraient être améliorées aujourd’hui, notamment sur la structure ou les bonnes pratiques, mais il représente une bonne première expérience en développement web.
